# Radically Straightforward · TSConfig

**✅ TypeScript configuration**

## Installation

```console
$ npm install @radically-straightforward/tsconfig
```

## Usage

`tsconfig.json`

```json
{
  "extends": "@radically-straightforward/tsconfig",
  "compilerOptions": {
    "rootDir": "./source/",
    "outDir": "./build/"
  }
}
```
